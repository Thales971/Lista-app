# Lista de Tarefas (React Native)

Aplicativo simples de lista de tarefas criado em **React Native (Expo)**.

## Funcionalidades

- Adicionar tarefas
- Remover tarefas tocando no botão **X**
- Lista atualizada em tempo real
- Mensagem quando não há tarefas

## Como rodar

1. Instale as dependências:

```bash
npm install
```

2. Inicie o Expo:

```bash
npm start
```

3. Abra no seu emulador ou no app Expo Go (Android/iOS) usando o QR code.

## Estrutura principal

- `App.js`: lógica principal da lista de tarefas
- `assets/`: imagens e recursos estáticos (caso sejam adicionados)

## Descrição para GitHub

Este repositório contém um exemplo de aplicativo de lista de tarefas desenvolvido com React Native e
Expo. O aplicativo permite adicionar, listar e remover tarefas de forma simples e rápida, sendo uma
boa base para quem está aprendendo a trabalhar com componentes, estado (`useState`) e listas
(`FlatList`).
